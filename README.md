# framework-bottleneck-calculator

A simple bottleneck calculator to test how good your Framework Laptop configuration is.

Project was entirely made in Next.JS. Some components are from Frameworks' home page. In general, the design is _strongly influenced_ by Frameworks' homepage.

This project was made for Hackclub Siege!

## Demo

Demo is live at https://framework-bottleneck-calculator.vercel.app/

## Building

Build this yourself by cloning the repository, installing the dependencies & building the app.

```bash
git clone https://github.com/RafaeloxMC/framework-bottleneck-calculator.git
cd framework-bottleneck-calculator
npm install
npm run build
```

Have fun!
